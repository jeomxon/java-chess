# 기능 요구사항

## 체스판

- [x] 말의 위치를 관리한다.
    - [x] 위치를 가진다.
    - [x] 말을 가진다.

- [x] 말의 초기 위치를 지정해준다.
- [x] 말의 초기 색을 지정해준다.

### 위치

- [x] 가로 위치(FILE)는 a ~ h로 표현된다.
- [x] 세로 위치(RANK)는 1 ~ 8로 표현된다.

## 말

- [x] 색을 가진다
    - [x] 흑 / 백

- [x] 종류가 있다
    - [x] 폰
    - [x] 나이트
    - [x] 비숍
    - [x] 룩
    - [x] 퀸
    - [x] 킹

## 체스게임

- [ ] 게임 시작과 종료를 제어한다.
